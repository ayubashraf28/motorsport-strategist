# game/sim

Authoritative deterministic simulation module used by the Godot runtime (V0 through V1.1).

This location is intentional for v0 because the Godot project root is `game/`.
See ADR `docs/adr/0001-v0-godot-root-and-sim-placement.md` for the extraction plan back to repo-level `sim/` when tooling/runtime constraints are relaxed.
